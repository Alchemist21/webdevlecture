Anatomy of a Complete Function



      Function Name: greet.

      Parameters: name, age

      Function Body: let message = ... and return message;

      Return Statement: Returns the message string

      // Function Declaration

      function greet(name, age) {

      // Function Body

      let message = "Hello, my name is " + name + " and I am " + age + " years old.";

      // Return Statement

      return message;

      }

      // Calling the function with arguments

      let result = greet("Alice", 25);

      console.log(result); // Output: Hello, my name is Alice and I am 25 years old.

Github



      GitHub is a cloud-based platform that lets you store, manage, and share code using Git, a version control system.

      git init – Initializes a new Git repository in your project folder.

      git add – Stages changes (files) to be included in the next commit.

      git commit – Saves a snapshot of the staged changes with a message

      git push – Uploads your commits to a remote repository like GitHub.:
